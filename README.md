# CatherineJun.github.io

Academic-style personal homepage for GitHub Pages.

## Files

- `index.html` — homepage content
- `style.css` — page styles

## Deploy

1. Create a GitHub repository named `CatherineJun.github.io`
2. Upload these files to the repository root
3. Wait a minute or two for GitHub Pages to publish
4. Visit: `https://catherinejun.github.io`

## Notes

- Update ORCID in `index.html` if needed
- Replace email if you want an anti-spam version such as `chenxiangjun [at] sjtu.edu.cn`
